# League of Legends Champions

React-приложение с информацией о чемпионах League of Legends. Данные
загружаются из официального Riot Data Dragon.

## Запуск

```bash
npm install
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`.

Команда `npm start` запускает тот же Vite dev-сервер.

## Команды

```bash
npm run dev
npm run build
npm run preview
npm run lint
```
